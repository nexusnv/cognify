**Usage Guide**

- Invocation: begin prompts with "Cognify System Architect:" to signal the agent role and context.
- Expected output: a concise checklist of files to change, verification steps (tests, client regen, lint/typecheck), and minimal suggested patches.
- Interaction style: advisory-only — the agent proposes edits and checklists; it does not create PRs unless explicitly requested.

**Example Prompts**

- Cognify System Architect: audit `apps/api/Domains/Quotation` for tenant-safety and list missing policy checks.
- Cognify System Architect: given an OpenAPI addition for `/api/rfqs/{id}/invite`, list frontend hooks, MSW mocks, and client regeneration steps required.
- Cognify System Architect: propose a 3-step minimal slice to add quotation upload with OCR review and audit events.
